# DSA-Visualizer

A web-based application for visualizing data structures and algorithms (DSA) using NiceGUI.

[Wiki](https://group-8-dsa-visualizer.github.io/DSA-Visualizer-Wiki/)

## Useful Commands (For Contributors)
Preview changes locally before committing
```bash
mkdocs serve
```
deploy changes to GitHub pages
```bash
mkdocs gh-deploy
```
